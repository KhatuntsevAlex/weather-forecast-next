# Security Policy

## Supported Versions

Only the latest released version on the `main` branch receives security updates.

## Reporting a Vulnerability

Please **do not** file public issues for security vulnerabilities.

Instead, report them privately via GitHub's
[Security Advisories](https://github.com/OWNER/REPO/security/advisories/new).

We aim to acknowledge reports within 72 hours and will coordinate a fix
and disclosure timeline with you.
